**Weather-Prediction-ML**

**Objective:** Develops a machine learning system to predict weather variables like temperature and rainfall using historical data.
**Overview:** Analyzes weather patterns to provide accurate forecasts for environmental monitoring and mission planning.

**Tasks**

**Data Preprocessing:** Cleaning raw weather datasets and handling missing values.
**Predictive Modeling:** Implementing Regression and Random Forest models for forecasting.
**Model Evaluation:** Calculating accuracy using MAE and RMSE metrics.
**Data Visualization:** Generating "Actual vs. Predicted" charts to verify performance.

**Technologies**

**Python**
**Pandas / NumPy**
**Scikit-learn**
**Matplotlib / Seaborn**
**Jupyter Notebook**

**Run Project**

```bash
pip install pandas scikit-learn matplotlib seaborn
jupyter notebook weather_analysis.ipynb

```
